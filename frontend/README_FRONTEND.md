# Wedding Guest Tracker - Frontend

React + TypeScript frontend for managing wedding guest lists.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Update API URL in `src/services/api.ts` if needed (default: `http://localhost:5001`)

3. Run development server:
```bash
npm run dev
```

Frontend runs on: http://localhost:5173

## Production Build

```bash
npm run build
```

Build output will be in `dist/` directory.

Preview production build:
```bash
npm run preview
```

## Features

- Dashboard with real-time statistics
- Family and member management
- Search and filter
- Full CRUD operations
- Responsive design
